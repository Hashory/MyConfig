# MyConfig

Management of personal configuration files (dotfiles) using [Dotter](https://github.com/SuperCuber/dotter).

## Installation

1.  [Install Dotter](https://github.com/SuperCuber/dotter#installation)
2.  Clone this repository:
    ```bash
    git clone https://github.com/hashory/MyConfig.git ~/program/MyConfig
    cd ~/program/MyConfig
    ```
3.  Deploy the configurations:
    ```bash
    dotter deploy
    ```
